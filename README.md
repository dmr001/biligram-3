# biligram
Version 3 - Vue JS

Daniel Rosenberg MD  
Providence St Joseph Health  
Portland, Oregon, USA

---

Plots newborn bilirubin nomograms for pre-discharge risk stratification and to help determine who needs treatment.
Implements:
- AAP 2004
- NCNC
- Stanford Premie BiliRecs thresholds
